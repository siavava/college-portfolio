# Personal Website[^1]

Built with [Nuxt 4](https://nuxt.com), [Vue 3](https://vuejs.org),
[Pug](https://pugjs.org), [Sass](https://sass-lang.com),
and [TypeScript](https://www.typescriptlang.org).

---

## Tech Stack

| Name | Description |
| :--- | :--- |
| [Nuxt 4](https://nuxt.com) | Full-stack Vue framework with SSR, file-based routing, and auto-imports |
| [Vue 3](https://vuejs.org) | Reactive component framework powering the UI layer |
| [Pug](https://pugjs.org) | Indentation-based HTML templating for cleaner Vue SFCs |
| [TypeScript](https://www.typescriptlang.org) | Typed superset of JavaScript for safer, self-documenting code |
| [Sass](https://sass-lang.com) | CSS preprocessor with variables, nesting, and mixins |
| [Nuxt Content](https://content.nuxt.com) | File-based CMS that turns Markdown/YAML into queryable collections |
| [Nuxt Icon](https://nuxt.com/modules/icon) | Icon component with access to 200k+ icons via Iconify |
| [VueUse](https://vueuse.org) | Collection of composable utilities (localStorage, clipboard, etc.) |
| [Vite](https://vitejs.dev) | Fast HMR dev server and optimized production bundler |
| [Vercel](https://vercel.com) | Deployment platform with edge rendering and analytics |

---

## Setup

Install dependencies:

```bash
bun install
```

### Development

Start the development server on http://localhost:3000:

```bash
bun dev
```

### Production

Build for production:

```bash
bun run build
```

Preview the production build locally:

```bash
bun preview
```

Generate static routes (recommended):

```bash
bun generate
```
